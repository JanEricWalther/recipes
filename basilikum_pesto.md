# Basilikum Pesto

## Zutaten (4 Portionen)

|           |             |
| --------: | ----------- |
|  3 Tassen | Basilikum   |
|    1 Zehe | Knoblauch   |
| 1/2 Tasse | Pinienkerne |
| 1/2 Tasse | Olivenöl    |
| 3/4 Tasse | Parmesan    |
|    1/4 TL | Salz        |

## Zubereitung

Alle Zutaten im Mixer zerkleinern, kühl aufbewahren.
